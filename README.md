🚀 USync

USync is a powerful, locally-hosted management dashboard for Android. Unlike traditional cloud-based sync tools, USync turns your phone into a local web server, allowing you to access your data directly from your computer without ever sending your files to an external server.

✨ Features

•📂 Professional File Browser: Full Import (Download) and Export (Upload) support. Navigate your phone's storage and transfer files seamlessly.

•💬 SMS Viewer: Read your recent text messages directly on your desktop.

•🔔 Notification Mirroring: Get real-time alerts from your phone applications in your browser.

•📍 Device Locator: Track your phone’s live GPS location on a map.

•📋 Smart Clipboard: Sync your phone's clipboard to your computer with a single click.

•📊 Device Overview: Monitor battery levels, storage usage, and system information.

•🔒 Privacy First: All data stays on your local Wi-Fi network. No cloud accounts or internet required.

🛠 Tech Stack

•Android: Java/Kotlin with Material 3 Design.

•Web Server: Embedded NanoHTTPD.

•Frontend: Pure HTML5, CSS3 (Modern UI), and Vanilla JavaScript.

•Permissions: Robust handling for Android 11+ (Scoped Storage, All Files Access, and Restricted Settings).


🚀 Getting Started

1.Clone the Repo: git clone https://github.com/YOUR_USERNAME/USync.git

2.Build the APK: Open the project in Android Studio and build the USync.apk

3.Run the App: Open USync on your phone and press Start Server.

4.Connect: Open the IP address shown on the screen (e.g., http://192.168.x.x:8080) in your computer's browser.


5.Sync: Ensure both devices are on the same Wi-Fi network.

📸 UI/UX Design

The app features a modern Material Design 3 interface with a clean, card-based layout, a scrolling marquee for long server addresses, and professional branding.
